# Pokédex

A simple web-based Pokédex application that allows you to search for Pokémon and view detailed information about them.

## Features

- 🔍 **Search Functionality**: Enter any Pokémon name to retrieve information
- 📱 **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- 🎨 **Modern UI**: Clean and user-friendly interface with gradient background
- 📊 **Comprehensive Information**: Displays the following details for each Pokémon:
  - **Name**: The official Pokémon name
  - **Type**: The elemental type(s) of the Pokémon
  - **Species**: Classification of the Pokémon
  - **Physiology**: Physical characteristics and appearance
  - **Behaviour**: Behavioral traits and personality
  - **Special Abilities**: Unique powers and capabilities
  - **Evolution**: Evolution information and chains

## How to Use

1. Open `index.html` in a web browser
2. Enter the name of a Pokémon in the search field
3. Click the "Search" button or press Enter
4. View the detailed information about the Pokémon

## Data Source

All Pokémon data is fetched from the [Pokémon Fandom Wiki](https://pokemon.fandom.com/) using the MediaWiki API.

## Technical Details

- **HTML**: Semantic markup structure
- **CSS**: Modern styling with flexbox and grid layouts
- **JavaScript**: Asynchronous data fetching and parsing
- **API**: MediaWiki API for accessing Pokémon Fandom wiki content

## Browser Compatibility

This application works in all modern browsers that support:
- ES6+ JavaScript
- Fetch API
- CSS Grid and Flexbox
- Local Storage (optional, for future enhancements)
